# 清理桌面缓存/临时文件

## 删除目标

共 6 项，按项目分组：

### 科技树\_系统工具（2 项）

- `科技树\_系统工具\tools\__pycache__\` — Python 编译缓存，48 KB，1 个 .pyc 文件
- `科技树\_系统工具\cursor_sync_temp\` — 旧版同步临时目录，96 KB，含 LOK/NCE/XS FABE 共 11 个废弃 MD（文件名含"废""废弃版""不看"等标记）

### 知识库及前端开发（4 项）

- `.cursor\debug-3f3d29.log` — Cursor 调试日志，14 KB
- `前端开发\data\execute.log` — 执行日志，19 KB
- `前端开发\data\dryrun.log` — 预演日志，19 KB
- `前端开发\node_modules\` — npm 依赖目录，20.4 MB / 225 个文件（可随时 `npm install` 重建）

## 执行方式

用 PowerShell `Remove-Item -Recurse -Force` 逐项删除，删除后确认每项不存在。

## 未触碰的文件

以下不在本次范围，保留不动：
- 各项目的 `cursor_excel/`、`cursor_sync/`、`cursor_docs/` 同步 MD
- `tools/*.py` 同步脚本和配置
- `.vscode/`、`.cursor/rules/` 等 IDE 配置
- `tessdata/`（OCR 训练数据，正常工具依赖）
