## 改前 vs 改后对比（以 P05「跨界迁移逻辑」为例）

### 改前（当前的样子，992 行）

每页含 5 大块视觉污染：

- ASCII 简笔版式预览（10+ 行）
- 结构槽位表（含字数建议 / kicker 颜色 / 字号）
- 起手稿 + 填空横线
- "硬钉子"标签
- 批注区

正文淹没在视觉描述里。

### 改后（目标）

```markdown
## P05 · 跨界迁移逻辑

> 这一页要讲：直面"为什么 ToB 光伏的人能做 ToC 消费电子"——这是评委心里最大的问号。

**为什么 ToB 光伏的人，能跑来做 ToC 消费电子？**

我的回答是——**我迁的不是品类，是一套"信息母本生产"的肌肉。**

- 在隆基：把组件原理、模拟软件、经济性测算说清楚，写成对外文案，再用中英文宣讲；
- 在韶音：把骨传导技术理解，写成 FABE、Slogan、Storytelling。

品类换了，行业换了，这套肌肉是通用的。
```

整页 = 标题 + 一句话定位 + 直接可读的正文。看完即可判断"这页讲清楚了没/这句话像不像我会说的"。

## 重写规则（19 页全部按这套来）

1. **保留**：页题（`## P0X · 页面名`）、`> 这一页要讲：…`（一句话定位，10-25 字）、正文
2. **删除**：ASCII 版式预览、结构槽位表、字数建议、kicker/字号/颜色描述、`[硬钉子]/[必填]/[建议]` 标签、填空横线、起手稿、每页底部的【批注】区
3. **正文写法**：
  - 直接是成品稿，不留空、不写"建议如此"
  - 来源 = `[讲稿.md](C:/Users/016551/OneDrive/Desktop/个人信息相关/转正答辩/讲稿.md)`，但按 PPT 节奏重新切分（讲稿是口播稿、PPT 是视读稿，不能 1:1 抄）
  - 表格类页（P08 / P14 / P17）保留 markdown 表格，但去掉"必填"占位符——直接填讲稿里有的内容；真正需要你提供的实际数字（双百积分、绩效）依然要留空，但用 `（待填实际数字）` 这种最小标注
4. **批注**：保留**全文末尾**一个统一的"全局批注区"即可，每页内不再放批注块——你想批注哪页直接在该页正文上写
