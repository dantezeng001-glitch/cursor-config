# 新内容归档计划

## 归档判断

我会按“先全量预检、再逐个归档、最后全库验收”的流程处理，避免再次只处理部分点名文件。

拟归档位置：

- `2025年度BHT报告展示版.pdf`：归到 `知识域/策略判断/用户调研/`，标准名建议为 `2025 年度 Shokz 品牌健康度调研（北美）`。
- `2026 Marketing策略沟通会.pdf`：归到 `知识域/策略判断/内部方法论/`，作为北美 MKT 策略原则与方法论资料。
- `2026年北美产品规划 20251127.pptx`：归到 `知识域/策略判断/历史规划/`，作为北美产品规划追溯资料。
- `AWE-2026洞察报告.pdf`：新增 `知识域/策略判断/外部研究/行业洞察/`，用于承接展会、行业趋势、消费电子观察类资料。
- `Shokz Brand Identity Guidelines V3.0 (EN).pdf`：新增 `知识域/策略判断/品牌资产/`，用于承接品牌视觉、品牌识别、品牌规范类资料。
- `SHOKZ北美高尔夫人群调研结论与洞察.md`：归到 `知识域/策略判断/用户调研/`。
- `Being 营销策略学习-太强了.png`：归到 `知识域/策略判断/历史规划/`，同时生成同名 Markdown 标准入口；图片作为源附件保留，若 OCR 可用则附 OCR 文本，否则先保留图片引用和人工可读摘要。

## 执行动作

- 先跑 `python 运行域/脚本/archive_inbox.py --dry-run`，并按用户给定路径逐个确认文件存在，防止目录扫描遗漏二进制或图片文件。
- 更新归档规则：在 `运行域/脚本/archive_inbox.py` 中补充 BHT、Marketing 策略、产品规划、AWE、Brand Identity、Golf、Being 等关键词规则。
- 更新结构说明：在 `运行域/治理文档/知识库架构说明.md` 和 `运行域/治理文档/命名与归档规范.md` 中补充 `品牌资产`、`行业洞察` 两个分类。
- 对 PDF/PPTX 走 `运行域/脚本/sync_binary_docs.py`，生成运行域同步 Markdown，并沉淀到知识域标准入口页。
- 对已有 Markdown 文件补齐 frontmatter、标准标题、`source_docs`、`canonical_path` 等字段。
- 对 PNG 单独生成标准入口页，并把图片源路径写入 `source_docs`；必要时把 `scan_untracked.py` 扩展为识别图片附件是否有同名 Markdown 入口。

## 验收

- 刷新 `运行域/治理文档/内容资料索引.md`。
- 追加 `运行域/治理文档/log.md`，记录本批归档、分类新增和同步状态。
- 跑 `archive_inbox.py --dry-run`，确认收件箱只剩维护文件。
- 跑 `scan_untracked.py`，确认无未登记文件、失效索引、收件箱业务残留、缺少标准 Markdown 入口的源文件。
- 对本次编辑的脚本和 Markdown 跑 linter 检查。
