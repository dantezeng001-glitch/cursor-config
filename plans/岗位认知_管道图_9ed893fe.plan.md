# 岗位认知 · 营销信息管道图

## 设计方案（已与用户确认的判断）

- 方向：**横向流体管道**（方向 A）
- 阀门拆成两道：**入口阀=主动收集防漏**、**出口阀=校验把关防失真**
- 主图结构：**3 条支线汇入 → 入口阀 → 主管道 → 母本罐体 → 主管道 → 出口阀 → 3 条支线扇出**
- 视觉记忆点：**贯穿不间断的管道**（强调"同一个母本在流动"）+ **中段膨大的罐体**（强调"被加工的容器"）+ **两个阀门**（强调 PMKT 的两道工序）
- 副标题升级：原 "PMKT 在营销信息链的上游" → **"PMKT 是营销信息管道的母本工程师"**

## 视觉规格（延续原图设计语言）

- 主色：黑 `#000` + 橙红 `#FF3D00` 类（强调）+ 浅灰底 `#F5F5F5`
- 字体：标题用粗黑体（思源黑体 Heavy / Microsoft YaHei Bold），正文用常规黑体
- 管道填充：左浅右深的橙→黑渐变，强化"母本沿管道传递"的方向感
- 罐体：椭圆胶囊形，深色填充 + 白字"母本"，比管道粗一圈，视觉上是"管道里的核心容器"
- 阀门：圆形 + 内部十字，带橙色高亮，下方一行小字说明"防漏 / 防失真"
- 留白：参考原图风格，整体 16:9，左右各留 60px 边距
