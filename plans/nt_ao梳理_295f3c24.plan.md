# NT-AO 工作逻辑梳理计划

## 初步判断

你这轮工作可以分成两段：旧阶段是“从 IMC 的 Open-Ear NR 技术命题出发，做一支 90 秒技术证明片”；新阶段是“从 NT 北美用户调研出发，反推人群、审美、片型和素材参考”。领导说旧创意“不 solid”，我的判断不是因为旧文件没做细，而是因为证据链主要压在工程参数和通用场景上，没有逐条绑定 NT 用户调研里的画像、卖点排序、场景和购买行为。

## 我会怎么整理

- 先做一页“你已经做了什么”：旧视频准备、新视频准备、调研源文档分别承担什么角色。
- 再做一页“工作逻辑”：`NT 调研 → 北美目标人群 → 创意偏好 → 参考素材 → 视频结构 → 渠道承接`。
- 再做一页“旧版为什么被认为不 solid”：不是否定旧版，而是标出它偏技术证明、偏通用场景、调研锚点不够显性。
- 再做一页“新方向应该怎么走”：用调研里的三组锚点压住创意，避免再变成空泛高级感。
- 最后列下一步产物：偏好卡、single-page treatment、45s/30s 脚本骨架、评审检查表。

## 建议的下一步默认产物

我建议先产出一份 `NT-AO新视频工作逻辑梳理_v0.1.md`，不要直接写完整脚本。它应包含四块：

- 当前问题复盘：旧创意为什么被认为不 solid。
- 调研锚点：北美 40+ 男性、品牌老用户/科技尝鲜、音质第一、滤噪是魅力属性、高频场景。
- 新创意方向：能力命名 + 声音可视化 + 滤噪开关对比 + 一场景一功能。
- 下一步任务单：偏好卡定稿、选 1 条主路线、落 45s/30s 脚本、建立评审检查表。
