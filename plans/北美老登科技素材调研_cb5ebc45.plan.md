## 一、目标与边界

- **做什么**：建一份素材库文档，供后续定"具体拍什么"时参考
- **不做什么**：不出脚本、不出分镜、不做创意概念（那是下一步）
- **风格基调锁定**：偏 B&O / Leica 式质感工业设计、品牌上层内容、低转化意图；刻意**排除**直给功能卖点的促销片、达人开箱、效果演示
- **源文档依据**：[营销传播活动/202604 NT-AO/NT上市后调研结论（截止亚马逊上市前）.md](营销传播活动/202604 NT-AO/NT上市后调研结论（截止亚马逊上市前）.md)

## 二、画像扩展（在 5 行基础上补 4 个维度）

**已从调研抽出的 5 行**：北美 40+ 商务男、品牌老用户追新（70%）、会主动搜第三方测评（~50%）、高频场景=工作/通话/家务/遛狗（非运动）、关键词音质>佩戴>环境声>滤噪。

**计划补的 4 维**（会在文档里标"我的判断"，不伪装规范）：

- **媒介习惯**：YouTube 测评频道（MKBHD/Linus/WSJ）、播客、长文（The Verge/Ars Technica）
- **审美参考系**：哪类品牌的工业设计 / 纪录片风格他们认——预判是 Apple / Leica / B&O / Teenage Engineering / Hermès 这条线
- **技术理解深度**：不是工程师、是"认参数但不较真"的爱好者（影响我们片子里术语的密度）
- **身份投射**：他们不是想"运动更强"，是想"被看作有品位的科技尝鲜者"——这决定片子的人物出场方式

## 三、调研三层结构（每层 10–12 条，合计约 30–36 条）

### A 层 · 同品类对标（耳机 / 音频）

候选品牌池（再筛选）：Sony（WF / LinkBuds / WH 系列）、Bose（QC Ultra / Ultra Open）、Apple（AirPods Pro / Max）、Bang & Olufsen、Sennheiser（Momentum / HD 系列）、Focal、Shure、Nothing Ear、Marshall、Master & Dynamic

**筛选规则**：只收各品牌近 2–3 年的**品牌片 / 产品工艺片 / 主题短片**；排除促销片、代言人带货、测评二创

### B 层 · 跨品类科技硬件（同画像）

候选品牌池：Apple（Vision Pro / Watch Ultra / iPhone Pro 产品片）、Meta Ray-Ban、Leica、Hasselblad、DJI（Osmo / Mavic）、Dyson、Tesla、Rolex、TAG Heuer、Rimowa、Teenage Engineering、Framework Laptop、Polestar、Porsche Design

**筛选规则**：看这些品牌怎么拍 **manifesto 片 / 发布片 / 工艺片**——这些就是老登心中"科技品牌的样子"

### C 层 · 纯视觉语言参考（执行手法库）

覆盖手法类型：

- CGI 剖切 / 材料特写（Apple Event 风）
- 慢镜头黑底产品蒙版
- 纪录片式工艺展示（Leica、B&O、Rimowa）
- 极简实拍 + 技术字幕（Teenage Engineering、Arc Browser）
- 声音可视化 / 抽象视觉化（Sonos、Sony Signature）

**筛选规则**：不关心品牌，只看**镜头语言 / 配乐节奏 / 字幕处理 / 剪辑密度**能不能抄

## 四、每条案例的字段（固定 5 项）

1. **品牌 + 片名 + 链接**（YouTube / Vimeo 为主）+ 时长
2. **为什么老登会看**（明确命中画像的哪一条）
3. **视觉手法标签**（CGI / 纪实 / 蒙版 / 声音可视化 / 极简字幕……）
4. **对 NT 可抄的一招**（一句话具体动作，不是"借鉴其极简风格"这种废话）
5. **预算感**（高 / 中 / 低；判断我们能不能真复刻）

## 五、调研方法与诚信底线

- 用 WebSearch + WebFetch 实搜 YouTube / Vimeo / 品牌官网，**不凭记忆写 campaign 名和链接**
- 每条链接标抓取日期；链接失效的标"待重找"而不是留死链
- 风格判断全部标"我的判断"，不写"业内公认 / 行业标杆"这类表述（`content-integrity` 纪律）
- 调研文档里的数字（70%、75%、50%+）若被引用到素材库文档，会在当前会话 Grep 自己确认没失真

## 七、交付节奏

- **第一版**：三层各先出 6–8 条 + 初步规律观察 → 你快速扫一遍确认方向
- **第二版**：补到每层 10–12 条 + 微调画像
- 如果第一版扫出来方向不对（比如你觉得 B&O/Leica 线不准），**退回本 plan 改范围**，不在成品上打补丁

## 八、风险点（提前摆在台面上）

- YouTube 区域锁：部分北美 campaign 国内抓不到详情 → 会标记"链接仅凭搜索结果"
- 品牌片时间久远的可能已下架 → 每条写抓取日期
- "为什么老登会看"带主观判断 → 会写理由链，你能逐条推翻
