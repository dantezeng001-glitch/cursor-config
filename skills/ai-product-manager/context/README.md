## 如何添加新公司

1. 在本目录下新建 `公司名.md`（如 `acme.md`）
2. 按以下结构填写内容：

```markdown
# [公司名] 业务上下文

## 品牌概览
- 品牌名、品类定位、市场布局

## 产品线
- 产品系列、代表产品

## 多语言体系（如适用）
- 核心语言、产品名规则

## 核心内容物
- 需要生产的内容类型及用途

## 关键业务流程
- 涉及的核心流程及挑战

## 已有工具/资产
- 现有的工具、知识库、自动化脚本等
```

3. 在 SKILL.md 的"业务上下文"段落中添加引用：`详见 [context/acme.md](context/acme.md)`

## 原则

- 每个公司一个独立文件，不要混在一起
- 文件内容保持事实性描述，不要包含操作指令
- 信息过时时直接更新对应文件即可
