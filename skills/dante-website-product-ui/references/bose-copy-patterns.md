# Bose Product Page — Copy Patterns

Premium-but-Warm：用第二人称感官词汇包裹技术参数，让高端产品没有距离感。

---

## 1. 产品名规范

- 完整名称含代数：`Bose QuietComfort Ultra Headphones (2nd Gen)`
- H1 展示完整产品名
- 正文中可缩写为品类名："the headphones" / "the earbuds"
- 评分紧贴 H1 下方（星级 + "rated X out of 5 by N"）

---

## 2. 叙事标题公式

4-8 个词的短句碎片，每个都是情感钩子。

| 公式 | 示例 |
|------|------|
| **短句碎片对** | "Seriously plush. Super stylish." / "Sleek meets soft" |
| **所有格 + 情感词** | "Our best noise cancelling" / "The realest feels" |
| **祈使句 / 邀请** | "Sink deeper into sound" / "Give your life a soundtrack" |
| **疑问 + 回答** | "Fit? Check." |
| **对比 / 反转** | "Hear every line. Feel everything you watch." |
| **升级式** | "The best. Now even better." / "The next level of legendary sound" |
| **感官隐喻** | "A hug for your ears" |

---

## 3. 产品定位首段公式

```
[情感定调] + [核心技术] + [核心体验] + [形态/设计] + [使用场景/结果]
```

**样本：**
> "Obsessively engineered with our best noise cancellation and spatialized audio for immersive listening, the QuietComfort Ultra Headphones (2nd Gen) make every note expand, every piece of dialogue hit harder, and every melody take up more space."

定位首段可折叠（"Read more" 展开），首屏仅展示首句。

---

## 4. 正文公式

1-3 句/段，第二人称贯穿。

### 语气特征

- **第二人称**：几乎每句都有 "you" / "your"
- **感官动词密集**："hit harder" / "sink deeper" / "feel" / "hear" / "goosebumps" / "chills"
- **技术表述柔化**：把参数包装成体验（"Up to 30 hours" → "Listen for longer than ever"）
- **Premium 但不距离感**：对话式句子消解高端产品的冰冷感
- **温暖**："hug for your ears" / "heart-stopping experience"

### 叙事区块正文结构

| 句序 | 作用 |
|------|------|
| 句1 | 情感/体验钩子（what it feels like） |
| 句2 | 技术支撑（how it works） |
| 句3（可选） | 场景或升级对比 |

## 6. CTA 文案

| CTA | 样式 | 位置 |
|-----|------|------|
| Add to Cart | 黑色实心按钮，白字 | 购买面板 + sticky 底部条 |
| View more | 文字链接 | Feature Pill 展开 |
| Read more | 文字链接 | 产品简介展开 |
| SHOW MORE / SHOW LESS | 全大写文字链 | Feature Card 底部 |
| Learn more > | 带箭头文字链 | 叙事区块内（仅部分） |
| Shop Now | 按钮 | Reviews 区底部 |

**策略**：
- 只有 "Add to Cart" 是实心按钮
- 叙事区块**零转化 CTA**——保持品牌沉浸感
- Sticky 底部条保证全程有购买入口

## 9. FAQ 文案

### Q 格式
第一人称或第三人称：
- "Can I use [Product] for phone calls?"
- "What is ActiveSense?"
- "How do I adjust the fit?"

### A 格式
第二人称（"your headphones"），教学语气，段落式详细解答。技术概念先给定义再讲工作原理。

---

## 10. 措辞禁区

- 叙事区块不放任何转化 CTA 按钮
- 不用冷硬技术术语做标题（用感官隐喻包装）
- 不用 "best in class" / "industry-leading"
- 不用感叹号做标题（正文偶尔可用）
- 链接不用蓝色（用黑色 + 下划线）

---

## 11. 叙事区块排列规律

每个产品页的品牌叙事区块按以下逻辑排序：

1. 总领升级定调（"The next level of…"）
2. 核心技术差异化（ANC / 空间音频 / OpenAudio）
3. 设计与舒适
4. 便捷功能（佩戴检测 / 充电 / 控制）
5. **最后一个区块统一为 Bose App 引导**（App Store + Google Play 链接）

## 12. 竞品文案洞察（来源：竞品 listing 分析）

以下洞察从 Bose 各产品线及同集团品牌（Jabra）listing 页提取，用于强化 Bose 风格文案的写法判断。

### 效果优先于参数（Effect Over Specs）

Bose 文案的底层哲学：不重点强调技术参数原理，重点宣传效果（沉浸）。这与 §4 正文公式中"技术表述柔化"一脉相承，但竞品分析进一步明确了优先级——**效果词永远排在参数词前面**。

- **写法指导**：每段声学正文的句1 必须是效果/体验句，句2 才放技术支撑。如果只有一句话的空间，只写效果，砍掉参数

### 隐形技术叙事（Invisible Technology Narrative）

原文："你听到的音乐就在眼前，你几乎感觉自己根本没有佩戴耳机。"这种"技术消失"的修辞让产品退居幕后，体验走到前台。

- **写法指导**：降噪和佩戴舒适段落，用"forget / disappear / vanish / 忘记 / 消失"类动词。目标是让读者觉得产品不存在——只有音乐存在

### 通感文案（Cross-Sensory / Synesthesia Copy）

原文："这是一种开创性的体验，声音如此真实，让你几乎忍不住想要伸出手去触摸它。"把听觉体验翻译成触觉冲动——Bose 独有的通感手法。

- **写法指导**：空间音频 / 沉浸式音频段落，至少嵌入一个跨感官描写（听→触 / 听→视 / 听→空间感）。避免停留在"听起来好"的单一维度

### 步骤化个性化（Step-by-Step Personalization）

CustomTune 文案："通过三个步骤定制声音。"把个性化过程拆成可数步骤，降低感知复杂度，同时提高用户参与感。竞品分析总结：通过提高参与感来增强对产品卖点的认同。

- **写法指导**：凡涉及 App 内操作或个性化功能，用"N 个步骤"结构。N ≤ 3 时参与感最强，N > 5 时反而增加感知负担

### 开放式耳机定位（Open Earbuds Positioning）

Ultra Open Earbuds 核心定位："音乐不停，畅听一切。"传递的核心价值是**不中断生活的音乐体验**。开放式音频的文案悖论：既是"专属于你的高质量声音"，又是"听到周围一切"。

- **写法指导**：开放式产品文案必须同时解决两个问题——①为什么音质不打折（"高质量"）②为什么开放是优势（"不错过"）。两者用"同时 / while / without losing"连接

### 中美文案策略差异

Bose 美国站在页面顶部嵌入第三方媒体评价引用（如 CNET、TechRadar），中国站不做此处理。这是区域信任建设策略的差异。

- **写法指导**：面向英文市场的 Bose 风格页面，在 Hero 区下方或产品简介上方预留一行 Review Quote 位置（斜体 + 来源标注）。中文市场可替换为 KOL 短评或电商评分

### Jabra 参考模式（同集团品牌）

Jabra 的两个文案手法可迁移至 Bose 风格：

| 手法 | 原文 | 迁移方式 |
|------|------|----------|
| 超大样本量佐证佩戴 | "62,000 挑一的完美贴合" | Bose 的舒适性声明可借鉴：用测试样本量（如"经 N 千种耳型测试"）代替主观形容词 |
| 云感舒适隐喻 | "云感舒适" | 符合 Bose 的感官隐喻传统（"A hug for your ears"），可在佩戴段落中作为备选隐喻 |
