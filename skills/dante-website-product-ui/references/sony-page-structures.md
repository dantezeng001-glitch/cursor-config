# Sony Style — Page Structures

Sony 将品牌叙事和电商转化拆成两套独立系统。

---

## A. 购买页（electronics.sony.com）

信息密集的电商转化页面，Tab 导航系统为核心骨架。

| 序号 | 区块 | 说明 |
|------|------|------|
| 1 | **购买面板** | 产品名 + 型号 + 价格 + 分期 + 颜色选择器 + ADD TO CART |
| 2 | **Feature Badges** | 横向滚动的特性徽章条（6-10 个核心卖点图标） |
| 3 | **Tab Nav** | About / Features / Specifications / Reviews / Questions / Support / Warranty |
| 4 | **Benefits Bar** | Easy Returns / My Sony Membership / Free Shipping / 0% APR / 5% Back |
| 5 | **Offers** | Trade-in / 订阅服务促销 |
| 6 | **About This Item** | 3 条核心卖点摘要（可展开） |
| 7 | **Features 深度区** | 媒体评价引用 + 技术分段（每段含三层叙事） |
| 8 | **Specifications** | 详细规格表（折叠式） |
| 9 | **Reviews & Questions** | UGC 评分 + 问答 |
| 10 | **Return Policy** | 退货政策 |
| 11 | **Frequently Bought Together** | 搭配推荐 |
| 12 | **Support / Warranty** | 支持链接 + 保修信息 |
| 13 | **Footnotes** | 大量脚注（30+ 条） |

### Feature Badges（Sony 独有）

购买页首屏下方放置横向滚动的特性徽章：
- 每个徽章 = 图标 + 1 行文字标题
- 8-10 个核心卖点，用户可快速扫读
- 样本：The Best Noise Cancellation / Extraordinary Sound Quality / Crystal Clear Call Quality / Up to 30-Hour Battery Life

---

## B. 营销页（sony.com 全球站）

长卷轴品牌叙事，无直接购买功能。可拆分为多个 Feature Sub-pages。

| 序号 | 区块 | 说明 |
|------|------|------|
| 1 | **Hero** | 产品名 + 核心 slogan + 暗色背景产品渲染 |
| 2 | **Overview 概述** | 1-2 段品牌叙事 + "Where To Buy" CTA |
| 3-N | **Feature Sections** | 每个核心卖点独立 section（暗底，全宽图文） |
| N+1 | **Design & Materials** | 设计语言 + 材质细节 |
| N+2 | **App / Connectivity** | Sound Connect App 功能 |
| N+3 | **Specifications** | 精简规格表 |
| N+4 | **Where To Buy** | 购买引导 |

### Feature Section 内部结构

```
[Section 标题 — 情感钩子]
[副标题 — 技术名称]
[正文 — 三层叙事]
[视觉 — 暗底全宽产品图/爆炸图]
[数据点 — 1-3 个量化对比]
[媒体评价引用 — 可选]
```

---

## C. 深浅节奏

Sony 以深色为主导（80%+ 面积），浅色段仅用于：
- Specifications 表格背景
- 搭配推荐区
- Footer / 脚注区

与 Apple 的 50/50 深浅交替形成鲜明对比。
