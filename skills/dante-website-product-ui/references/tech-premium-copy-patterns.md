# Tech Premium Style — Copy Patterns

> 近似品牌：Sony

## Headline 公式

### 三层叙事（Sony 签名手法）

每个 feature section 遵循三层结构：

| 层 | 作用 | 样本（WH-1000XM6） |
|----|------|---------------------|
| 1. 情感钩子 | 感性短句引起共鸣 | "The next evolution in noise cancellation" |
| 2. 技术实体 | 点名具体芯片/零件型号 | "HD Noise Canceling Processor QN3 — 7x faster than QN1" |
| 3. 量化对比 | 与前代/竞品数字对比 | "1.5x more noise cancelled vs XM5" |

### Headline 句式样本

| 产品 | Headline |
|------|----------|
| WH-1000XM6 | "Redefine silence. Redefine sound." |
| WH-1000XM5 | "Magnificent sound, engineered to perfection" |
| WF-1000XM5 | "The best noise canceling earbuds" |
| LinkBuds S | "Small and light for all-day comfort" |

特点：比 Apple 更直白，直接宣称"best"/"redefine"；比 Beats 更克制，不用口语。

## 正文公式

- 第一句：技术名称 + 代际进化（"HD Noise Canceling Processor QN3, with 7x the processing power of QN1"）
- 第二句：技术原理简述（"fine-tunes 12 microphones in real time"）
- 第三句：用户体验结果（"delivering a dramatic leap in noise cancellation"）

## 技术术语规范

Sony 喜欢给技术加 "™" 和代际编号：
- DSEE Extreme™
- Dynamic Driver X
- Integrated Processor V2
- HD Noise Canceling Processor QN3
- Noise Isolation Earbud Tips

## 背书模式

Sony 是所有品牌中最重背书的：
- 媒体评分引用：CNET / CNN Underscored / TechRadar（附 logo + 评分 + 短评）
- 合作音频工程师/艺人证言
- Hi-Res / LDAC 认证 logo 展示
- "Walkman 工艺传承" 叙事

## CTA 用词

| 位置 | 文案 |
|------|------|
| 购买页 | ADD TO CART |
| 营销页 | Where To Buy / Learn More |
| 对比表 | Compare / See All |
| App | Download App |

## 措辞特点 vs 其他风格

- 比 Apple 更技术：直接写芯片型号、频响范围、dB 数值
- 比 Chinese Tech 更克制：不堆砌数据，选最关键的 3-4 个量化点
- 比 Bose 更理性：少用感官词，多用工程词
- 独有的"Walkman 工艺传承"叙事：含金焊料、镀铜电阻等细节

## Excel 竞品调研 insight 补充

- 索尼"源自 Walkman 工艺"的表达方式：通过工艺传承叙事提升品牌信任感
- DSEE Extreme 与 Hi-Res 结合讲解：技术功能 + 认证背书 = 双重说服力
- 360 临场音效表达："沉浸在全方位包裹的音乐中，好似亲临现场音乐会"
- 降噪文案结构：芯片算力（速度/深度）+ 麦克风数量/排布（精准度）+ 算法（自适应）+ 被动降噪（耳塞/耳垫）
