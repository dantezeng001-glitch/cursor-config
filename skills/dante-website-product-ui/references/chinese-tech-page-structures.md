# Chinese Tech Style — Page Structures

基于 OPPO / Huawei / Soundcore / Oladance 跨品牌提取的共性页面结构。

---

## 核心架构：一功能一全屏

中国品牌产品页的标志性布局——每个核心卖点占据一个全屏高度的 section，用户逐屏滚动消费内容。

---

## 完整区块序列

| 序号 | 区块 | 说明 |
|------|------|------|
| 1 | **Dark Hero** | 暗色全宽背景 + 悬浮产品渲染图 + 产品名 + 核心 slogan |
| 2 | **技术标签条** | 5-8 个核心技术名横排（如 LHDC 4.0 / 骨传导 / 双单元 / ANC） |
| 3 | **奖项/联名条** | CES / 红点 / 联名品牌 logo（如有） |
| 4-N | **Feature Sections** | 每个核心卖点 = 一个全屏暗底 section（见下方详细结构） |
| N+1 | **数据对比面板** | 与上代产品的 4-6 组数据并排对比（大字号 + 条形图） |
| N+2 | **设计/材质** | 外观设计 + 配色展示 + 材质工艺细节 |
| N+3 | **续航** | 大数字 stat callout + 快充数据 + 充电方式 |
| N+4 | **次要功能合并区** | 3-6 个辅助功能以卡片/grid 形式合并展示 |
| N+5 | **App** | 应用功能展示 + 下载引导 |
| N+6 | **Specifications** | 详细规格表（展开式） |
| N+7 | **What's in the Box** | 配件清单（部分品牌有） |
| N+8 | **FAQ** | 手风琴式 FAQ（Soundcore 有，其他品牌少见） |
| N+9 | **Footnotes** | 极其详细的脚注（15-18 条） |

---

## Feature Section 内部结构

```
┌─────────────────────────────────────────────┐
│  [暗色全宽背景]                               │
│                                             │
│  [技术名称 + 版本号]  ← 18-24px, 半透明      │
│  [大标题：感性短句]   ← 36-48px, 白色, bold   │
│                                             │
│  [正文：工程叙事 2-3 句] ← 16px, 灰色        │
│                                             │
│  ┌──────┐  ┌──────┐  ┌──────┐              │
│  │ 数据1 │  │ 数据2 │  │ 数据3 │  ← 大字号   │
│  │ 标签  │  │ 标签  │  │ 标签  │  ← 小字    │
│  └──────┘  └──────┘  └──────┘              │
│                                             │
│  [产品爆炸图/结构图/场景图]                    │
│                                             │
└─────────────────────────────────────────────┘
```

## 数据对比面板（中国品牌独有）

与前代产品的并排对比是核心叙事手法：

```
┌─────────────────────────────────────┐
│        vs 上代                       │
│  ┌─────────┐    ┌─────────┐        │
│  │  2x     │    │  +67%   │        │
│  │ 采样率   │    │  磁铁   │        │
│  │ [前代条] │    │ [前代条] │        │
│  │ [本代条] │    │ [本代条] │        │
│  └─────────┘    └─────────┘        │
└─────────────────────────────────────┘
```

## 深浅节奏

中国品牌以深色为绝对主导（90%+ 面积），浅色仅用于：
- Specifications 表格
- FAQ 区域
- Footnotes

背景色通常为纯黑 #000000 或极深灰 #0a0a0a，文字为白色 #ffffff 或浅灰 #cccccc。
